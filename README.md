 ## 
  
 ### Installation

- Clone the repository
  
  ```
  git clone https://github.com/SamueleAmato/Archiver-Z
  ```
- Go to cloned directory and run ```az.sh```
  
  ```
  cd Archive-Z
  bash az.sh
  ```
- On first launch, It'll install the dependencies.
